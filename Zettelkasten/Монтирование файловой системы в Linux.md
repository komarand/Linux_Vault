# Монтирование файловой системы в Linux

В Linux файловые системы физических дисков должны быть "встроены" (смонтированы) в единое логическое дерево каталогов, чтобы к ним можно было получить доступ.

Команда `mount` связывает устройство (например, `/dev/sdb1`) с существующим пустым каталогом, называемым "точкой монтирования" (например, `/mnt/usb`). Все файлы и каталоги на диске становятся доступными через этот путь.

Процесс отключения называется размонтированием (`umount`) и должен быть выполнен перед физическим извлечением носителя, чтобы гарантировать запись всех закэшированных данных на диск.
