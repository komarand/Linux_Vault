# Права доступа к файлам в Linux

Каждый файл и каталог в Linux имеет владельца (пользователя), группу-владельца и три набора разрешений: для владельца (u), группы (g) и остальных (o).

Разрешения бывают трех видов:
- Чтение (`r` = 4): для файлов — чтение содержимого, для каталогов — просмотр списка файлов (`ls`).
- Запись (`w` = 2): для файлов — изменение, для каталогов — создание и удаление файлов в нем.
- Выполнение (`x` = 1): для файлов — запуск программы или скрипта, для каталогов — право входа в него (`cd`) и доступ к его содержимому.

Права обычно записываются в виде 10-символьной строки (например, `-rwxr-xr--`) или в восьмеричной системе (например, `754`).
