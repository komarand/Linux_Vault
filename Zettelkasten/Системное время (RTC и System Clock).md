# Системное время (RTC и System Clock)

В Linux существует два понятия времени:

1. **Аппаратные часы реального времени (RTC или Hardware Clock)**: энергонезависимые часы на материнской плате (с батарейкой), которые идут даже при выключенном компьютере. Используются загрузчиком и ядром при старте.
2. **Системные часы (System Clock)**: программные часы внутри ядра Linux, отсчитывающие время (в секундах) с "эпохи Unix" (1 января 1970 г.).

При загрузке ядро считывает время из RTC и устанавливает системные часы. Далее операционная система использует только системные часы, корректируя их точность по протоколу NTP.
