# Перемещение по каталогам cd

Команда `cd` (change directory) используется для изменения текущей рабочей директории в оболочке.

Это встроенная команда оболочки, а не внешняя программа.

Специальные символы в путях для `cd`:
- `.` — указывает на текущий каталог.
- `..` — указывает на родительский каталог (на один уровень вверх).
- `~` — указывает на домашний каталог текущего пользователя.
- `-` — переключает в каталог, в котором пользователь находился до последнего использования команды `cd`.

Эти сокращения ускоряют навигацию по файловой системе.
