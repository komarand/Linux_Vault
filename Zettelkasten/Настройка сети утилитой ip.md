# Настройка сети утилитой ip

Утилита `ip` из пакета `iproute2` — это современный швейцарский нож для настройки сети в Linux, заменяющий старые команды `ifconfig` и `route`.

Основные функции:
- `ip link`: просмотр и управление состоянием интерфейсов (включить/выключить через `ip link set eth0 up`).
- `ip addr`: управление IP-адресами интерфейсов.
- `ip route`: просмотр и настройка таблицы маршрутизации (включая шлюз по умолчанию — default gateway).
- `ip neigh`: просмотр ARP-таблицы (соответствие IP и MAC-адресов).
Изменения, сделанные командой `ip`, работают только до перезагрузки.
