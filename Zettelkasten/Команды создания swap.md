# Команды создания swap

Для настройки области подкачки используются три основные команды:

1. `mkswap`: инициализирует (форматирует) раздел или файл в качестве структуры подкачки. Пример: `mkswap /dev/sda2`.
2. `swapon`: активирует область подкачки, сообщая ядру, что теперь ее можно использовать для выгрузки страниц памяти. Пример: `swapon /dev/sda2`.
3. `swapoff`: деактивирует (отключает) область подкачки. Перед выполнением ядро должно переместить все страницы из `swap` обратно в физическую ОЗУ (если ее достаточно).

Для автоматического включения `swap` добавляется в файл `/etc/fstab`.
