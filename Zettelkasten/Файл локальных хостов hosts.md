# Файл локальных хостов hosts

Файл `/etc/hosts` используется для локального (ручного) разрешения доменных имен в IP-адреса до обращения к DNS-серверам.

Каждая строка содержит IP-адрес и соответствующее ему имя (или несколько имен). Например: `127.0.0.1 localhost`.
Это крайне полезно для разработчиков и сисадминов: можно перенаправить домен на тестовый локальный сервер или заблокировать рекламные серверы, направив их домены на `127.0.0.1` (или `0.0.0.0`), не дожидаясь обновления глобальных DNS-записей.
