# Файловые системы ext4, xfs, btrfs

После создания раздела, для хранения файлов его необходимо отформатировать в определенную файловую систему (ФС).

Основные ФС в Linux:
- **ext4**: стандартная и самая проверенная ФС по умолчанию в большинстве дистрибутивов. Поддерживает журналирование для восстановления после сбоев.
- **XFS**: высокопроизводительная 64-битная журналируемая ФС от SGI. Отлично подходит для огромных файлов и баз данных (часто используется по умолчанию в RHEL/CentOS).
- **Btrfs** (B-tree FS): современная ФС следующего поколения (используется в SUSE, Fedora). Поддерживает создание снимков (snapshots), дедупликацию, проверку контрольных сумм данных и встроенный менеджер томов.
