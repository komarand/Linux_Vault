# Временная файловая система initramfs

Образ `initramfs` (initial RAM file system) — это сжатый архив, загружаемый в оперативную память вместе с ядром.

Он содержит минимальный набор утилит и драйверов (модулей ядра), необходимых ядру для доступа к реальной корневой файловой системе на диске.

Например, если корневая ФС находится на LVM-томе с шифрованием LUKS, самому ядру не хватает встроенных средств для их монтирования. `initramfs` содержит утилиты LVM и ключи расшифровки. После монтирования реального корня скрипты `initramfs` передают управление настоящему процессу `init` на диске.
