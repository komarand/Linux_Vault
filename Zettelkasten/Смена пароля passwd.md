# Смена пароля passwd

Утилита `passwd` используется для изменения пароля пользователя.

При выполнении обычным пользователем она запрашивает старый пароль, а затем дважды новый. Пароли в Linux хранятся не в открытом виде, а в виде хэшей, обычно в системном файле `/etc/shadow`.

Суперпользователь (root) может использовать команду `passwd username` для принудительной смены пароля любого пользователя, не зная его текущего пароля.
