# Кэширование файловой системы

Дисковые операции ввода-вывода — самые медленные процессы в компьютере. Для ускорения работы ядро Linux использует механизм Page Cache (кэш страниц).

Когда вы читаете файл, ядро копирует его блоки в оперативную память. Если вы или другая программа прочитаете этот файл снова, ядро отдаст его из ОЗУ (со скоростью памяти), не обращаясь к диску.

При записи файлов данные сначала пишутся в кэш ОЗУ (становятся "грязными" страницами - dirty pages). Позже ядро асинхронно сбрасывает их на диск. Это сильно ускоряет запись, но создает риск потери данных при внезапном отключении питания.
