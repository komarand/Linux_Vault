# Скрытые файлы в Linux

В Linux любой файл или каталог, имя которого начинается с точки (например, `.bashrc`, `.config/`), считается скрытым.

Они не имеют специальных атрибутов файловой системы, скрывающих их (как в Windows); утилиты вроде `ls` или графические файловые менеджеры просто игнорируют их по умолчанию.

Скрытые файлы почти всегда используются для хранения конфигураций и настроек пользователя. В домашнем каталоге их называют "файлы конфигурации" (dotfiles). Для их отображения в консоли используется команда `ls -a`.
