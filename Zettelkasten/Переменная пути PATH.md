# Переменная пути PATH

Переменная окружения `PATH` — одна из важнейших переменных в Linux.

Она содержит разделенный двоеточиями список каталогов, в которых оболочка ищет исполняемые файлы (программы) при вводе команды. Если вы вводите `ls`, оболочка не ищет по всему диску; она проверяет каталоги из `PATH` (например, `/bin`, `/usr/bin`).

Если исполняемый файл не находится в каталоге, включенном в `PATH`, для его запуска необходимо указать точный путь, например абсолютный (`/opt/myapp/bin/app`) или относительный (`./app` в текущей директории).
