# IP-адреса и интерфейсы

В Linux сетевые интерфейсы (физические сетевые карты или виртуальные адаптеры) обозначаются именами, такими как `eth0` (Ethernet), `wlan0` (Wi-Fi), или современными предсказуемыми именами (Predictable Network Interface Names) вроде `enp3s0`.

Специальный виртуальный интерфейс `lo` (loopback) с IP-адресом `127.0.0.1` используется для связи программ внутри самого компьютера.
Для просмотра и настройки интерфейсов и IP-адресов используется современная утилита `ip addr` (или `ip a`). Устаревшая утилита `ifconfig` больше не входит в базовую поставку большинства дистрибутивов.
