# Таблица fstab

Файл `/etc/fstab` (file system table) содержит список файловых систем и параметров их монтирования, которые система должна автоматически подключать при загрузке.

Каждая строка описывает одну ФС в 6 полях:
1. Устройство (путь к блочному устройству или UUID).
2. Точка монтирования в иерархии.
3. Тип файловой системы (ext4, vfat, xfs).
4. Параметры монтирования (ro, defaults, noexec).
5. Опция dump (часто 0).
6. Порядок проверки fsck при загрузке (1 для корневой ФС, 2 для остальных, 0 - не проверять).

Ошибки в `fstab` могут привести к сбою загрузки системы в режим восстановления.
