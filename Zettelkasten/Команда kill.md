# Команда kill

Вопреки своему названию, команда `kill` предназначена не только для "убийства" процессов, а для отправки им любых сигналов.

По умолчанию `kill PID` отправляет сигнал `SIGTERM` (15), позволяя процессу завершиться корректно.
Если процесс "завис" и не реагирует на `SIGTERM`, используется принудительное завершение: `kill -9 PID` (отправка `SIGKILL`).

Также существуют утилиты `killall` (отправляет сигнал всем процессам с указанным именем) и `pkill` (отправляет сигнал процессам, чьи имена совпадают с регулярным выражением).
