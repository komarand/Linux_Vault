# Просмотр текстовых файлов less

Команда `less` — это пейджер (программа для просмотра текста по страницам), который позволяет просматривать содержимое больших текстовых файлов без загрузки всего файла в память.

В отличие от старой программы `more`, `less` позволяет прокручивать текст не только вперед, но и назад. Она предоставляет возможности поиска по тексту внутри файла (`/` для поиска вперед, `?` для поиска назад), перехода к конкретным строкам и удобной навигации клавишами со стрелками или Page Up / Page Down.

Для выхода из программы нажимается клавиша `q`.
