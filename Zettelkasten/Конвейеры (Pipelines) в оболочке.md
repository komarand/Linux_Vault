# Конвейеры (Pipelines) в оболочке

Конвейер, обозначаемый символом вертикальной черты (`|`), позволяет связать несколько команд в цепочку.

Стандартный вывод (stdout) первой команды передается напрямую в стандартный ввод (stdin) второй команды, минуя создание временных файлов. Это позволяет комбинировать простые утилиты для выполнения сложных задач обработки данных.

Пример: `cat file.txt | grep "error" | sort | less`. Эта цепочка читает файл, отбирает строки с ошибками, сортирует их и выводит результат в программу для постраничного просмотра.
