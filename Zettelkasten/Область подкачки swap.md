# Область подкачки swap

Область подкачки (swap space) — это часть дискового пространства, которую ядро Linux использует в качестве расширения оперативной памяти (виртуальная память).

Когда физическая ОЗУ заполняется, ядро перемещает неактивные страницы памяти процессов на диск в `swap`, освобождая ОЗУ для более приоритетных задач. Процесс перемещения называется свопингом. Поскольку диски работают в тысячи раз медленнее ОЗУ, активный свопинг (thrashing) приводит к катастрофическому падению производительности (тормозам).

Swap может быть организован как в виде выделенного раздела, так и в виде файла подкачки (swap file).
