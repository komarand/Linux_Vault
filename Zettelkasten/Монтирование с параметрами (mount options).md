# Монтирование с параметрами (mount options)

При монтировании файловых систем с помощью `mount -o` или через `fstab` можно задавать специальные параметры, влияющие на поведение или безопасность:

- `ro` (read-only): монтирование только для чтения (запрет любых модификаций).
- `rw` (read-write): стандартное монтирование с возможностью чтения и записи.
- `noexec`: запрет выполнения любых исполняемых файлов или скриптов на этой ФС (полезно для `/tmp` или пользовательских накопителей).
- `nosuid`: игнорирование бита SUID, предотвращающее повышение привилегий через запуск файлов (важно для переносных носителей).
