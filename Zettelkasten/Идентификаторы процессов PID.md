# Идентификаторы процессов PID

Каждому процессу в Linux при создании ядро назначает уникальный числовой идентификатор — PID (Process ID).

PID используется ядром и пользовательскими утилитами (такими как `kill`, `strace`, `top`) для точного обращения к конкретному запущенному экземпляру программы.
PID 1 всегда резервируется для процесса инициализации (`init` или `systemd`).
Помимо своего PID, каждый процесс знает идентификатор своего процесса-родителя — PPID (Parent Process ID). Это формирует иерархическое дерево процессов.
