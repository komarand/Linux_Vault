# Загрузчик GRUB

GRUB (GRand Unified Bootloader) версии 2 — стандартный и наиболее функциональный загрузчик в мире Linux.

Он предоставляет меню для выбора операционной системы или версии ядра при включении компьютера. Главный конфигурационный файл GRUB находится по пути `/boot/grub/grub.cfg`, однако он не редактируется напрямую.

Вместо этого администраторы вносят изменения в файл `/etc/default/grub` или скрипты в `/etc/grub.d/`, а затем генерируют новый конфигурационный файл с помощью утилиты `grub-mkconfig` (в Debian/Ubuntu — `update-grub`).
