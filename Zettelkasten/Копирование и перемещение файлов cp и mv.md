# Копирование и перемещение файлов cp и mv

В Linux для управления файлами используются две основные команды: `cp` (copy) и `mv` (move).

- Команда `cp` копирует файлы или каталоги. Для копирования каталогов вместе с их содержимым используется рекурсивный флаг `-r`. Оригинал файла остается на месте.
- Команда `mv` перемещает файлы из одного расположения в другое. Оригинал удаляется из исходной директории. `mv` также используется для переименования файлов и каталогов, поскольку переименование в Linux — это фактически перемещение в ту же директорию под новым именем.
