# Виртуальные терминалы tty

Файлы `/dev/ttyX` (например, `/dev/tty1`, `/dev/tty2`) представляют виртуальные терминалы (консоли).

Исторически `tty` означало teletypewriter. Linux предоставляет несколько виртуальных консолей, переключаться между которыми можно с помощью `Ctrl+Alt+F1`...`F6`. Они позволяют работать в текстовом режиме без графической оболочки.

Терминалы, открываемые в графическом окружении (например, эмуляторы терминала GNOME Terminal) или по SSH, используют псевдотерминалы, которые обозначаются как `/dev/pts/0`, `/dev/pts/1` и т.д.
