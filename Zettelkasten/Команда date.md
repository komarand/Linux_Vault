# Команда date

Утилита `date` используется для вывода текущего системного времени и даты в терминале, а также (с правами root) для их ручной установки.

Она часто применяется в скриптах оболочки для генерации имен файлов или логов с временными метками.

Вывод команды можно гибко форматировать с помощью спецификаторов:
Пример: `date +"%Y-%m-%d_%H-%M-%S"` выведет строку в формате `2023-10-25_14-30-00`.
Для просмотра аппаратного времени (RTC) используется отдельная утилита `hwclock`.
