# Суперпользователь root

Пользователь с именем `root` (UID 0) — это суперпользователь, администратор системы.

Ядро предоставляет процессам, запущенным от имени root, полные, неограниченные привилегии. Root может читать, записывать и удалять любой файл в системе, изменять конфигурации, запускать порты ниже 1024, и управлять процессами других пользователей.

Из-за таких широких полномочий постоянная работа под учетной записью root не рекомендуется. Для выполнения административных задач предпочтительно использовать утилиту `sudo`, которая временно повышает привилегии.
