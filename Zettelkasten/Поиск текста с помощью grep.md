# Поиск текста с помощью grep

Команда `grep` — мощный инструмент для поиска текстовых строк или шаблонов (регулярных выражений) внутри файлов или в потоке ввода.

Она построчно проверяет ввод и выводит только те строки, которые соответствуют заданному шаблону.

Ключевые флаги:
- `-i`: игнорировать регистр.
- `-v`: инвертировать поиск (выводить строки, НЕ содержащие шаблон).
- `-r` или `-R`: рекурсивный поиск в каталоге.
- `-n`: выводить номера строк, где найдено совпадение.

`grep` часто комбинируется с другими командами через конвейер (pipeline) для фильтрации вывода.
