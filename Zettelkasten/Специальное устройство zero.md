# Специальное устройство zero

Устройство `/dev/zero` при чтении из него бесконечно генерирует нулевые байты (не ASCII-символ '0', а байт со значением 0x00).

Оно часто применяется в комбинации с утилитой `dd` для создания пустых файлов заданного размера (например, для файлов подкачки или образов виртуальных машин) или для затирания данных на дисках (перезапись нулями).

Пример: `dd if=/dev/zero of=large_file.img bs=1M count=1024` создаст файл размером 1 ГБ, заполненный нулями.
