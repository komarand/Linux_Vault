# Переменные окружения и оболочки

Оболочка поддерживает переменные для хранения текстовой информации (пути, настройки). Существует два типа:

1. **Переменные оболочки**: локальны для текущего экземпляра оболочки (процесса). Они не передаются программам, которые запускаются из этой оболочки.
2. **Переменные окружения (Environment variables)**: экспортируются дочерним процессам. Программы могут считывать их для изменения своего поведения (например, язык интерфейса или пути конфигурации).

Команда `export` превращает локальную переменную оболочки в переменную окружения. Команда `env` выводит список всех переменных окружения.
