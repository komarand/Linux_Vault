# Создание пустых файлов touch

Команда `touch` в Linux используется для создания пустых файлов или обновления временных меток (timestamp) существующих файлов.

Если передать команде `touch` имя несуществующего файла, она создаст пустой файл с этим именем. Если файл уже существует, `touch` по умолчанию обновит его время доступа и время модификации до текущего системного времени, не изменяя содержимого.

Это часто применяется для создания файлов-заглушек или запуска процессов резервного копирования (backup), которые зависят от времени модификации.
