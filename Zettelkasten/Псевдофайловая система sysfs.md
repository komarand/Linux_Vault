# Псевдофайловая система sysfs

Псевдофайловая система `sysfs` смонтирована в каталоге `/sys` и предоставляет пользовательскому пространству представление ядра о подключенном оборудовании.

В отличие от `/dev`, который содержит файлы для обмена данными с устройствами, `/sys` используется для конфигурации устройств, считывания их статуса (например, температуры, скорости вращения кулера) и получения информации об иерархии (какое устройство к какому контроллеру подключено). Данные генерируются ядром "на лету" при обращении к файлу.
