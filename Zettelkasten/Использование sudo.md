# Использование sudo

Утилита `sudo` (substitute user do) позволяет обычным пользователям выполнять команды с правами суперпользователя (`root`) или другого пользователя.

Когда пользователь вызывает команду с префиксом `sudo`, он вводит свой собственный пароль, а не пароль root. Настройка прав (каким пользователям разрешено запускать какие команды) осуществляется в файле `/etc/sudoers`.

Использование `sudo` оставляет след в системных журналах, что улучшает безопасность и аудит действий администраторов по сравнению с прямым входом под пользователем `root`.
