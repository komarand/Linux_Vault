# Определение типа файла file

Команда `file` анализирует содержимое файла для определения его типа.

В отличие от Windows, Linux не полагается исключительно на расширения файлов (например, `.txt`, `.exe`) для понимания типа файла. Утилита `file` читает заголовок файла или его структуру (магические числа — magic numbers) и описывает, чем он является (например, текстовый файл ASCII, исполняемый файл ELF, архив gzip, изображение JPEG).

Это помогает безопасно и корректно обращаться с неизвестными файлами.
