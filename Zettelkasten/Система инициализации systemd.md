# Система инициализации systemd

`systemd` — это современная система инициализации и менеджер служб (service manager), ставшая стандартом де-факто в Linux (заменяющая SysVinit).

Она запускается с PID 1 и управляет загрузкой системы.
Особенности `systemd`:
1. Параллельный запуск служб (демонов) для ускорения загрузки.
2. Управление зависимостями (служба сети запускается строго до веб-сервера).
3. Мониторинг и автоматический перезапуск упавших демонов.
4. Унифицированное журналирование (systemd-journald).
5. Использование конфигурационных файлов (`.service`), называемых "юнитами" (units).
