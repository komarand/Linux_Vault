# Команда ls

Команда `ls` используется для вывода содержимого каталога. По умолчанию она отображает имена файлов и подкаталогов в текущей рабочей директории.

Основные флаги `ls`:
- `-l`: длинный формат вывода. Показывает права доступа, владельца, группу, размер файла и время последней модификации.
- `-a`: показывать все файлы, включая скрытые (файлы, имена которых начинаются с точки).
- `-h`: выводить размер файлов в удобочитаемом формате (КБ, МБ, ГБ), работает совместно с флажком `-l`.

Это основной инструмент для изучения содержимого файловой системы.
