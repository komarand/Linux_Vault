# Создание файловой системы mkfs

Команда `mkfs` (make file system) используется для форматирования дискового раздела (создания на нем файловой системы).

На самом деле `mkfs` — это оболочка (фронтэнд), которая вызывает конкретную программу форматирования в зависимости от типа ФС, например `mkfs.ext4`, `mkfs.xfs` или `mkfs.vfat`.

Пример: `mkfs -t ext4 /dev/sdb1` отформатирует первый раздел второго диска в файловую систему ext4. Эта операция уничтожит все старые данные и структуру на разделе, создав новые таблицы `inode` и суперблок.
