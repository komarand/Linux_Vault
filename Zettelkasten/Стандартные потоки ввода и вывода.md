# Стандартные потоки ввода и вывода

В Linux взаимодействие с программами организовано через стандартные потоки (streams). Существует три основных потока:

1. **Стандартный ввод (stdin)**: поток данных, поступающий в программу. Обычно связан с клавиатурой, но может быть перенаправлен из файла или другой программы.
2. **Стандартный вывод (stdout)**: поток нормальных данных, исходящих из программы. Обычно выводится на экран терминала.
3. **Стандартная ошибка (stderr)**: отдельный поток для сообщений об ошибках и диагностической информации.

Разделение потоков `stdout` и `stderr` позволяет перенаправлять полезные данные в файл, сохраняя сообщения об ошибках на экране, что критически важно для создания скриптов и конвейеров (pipelines).
