# Перенаправление ввода-вывода

Оболочка позволяет перенаправлять стандартные потоки в файлы с помощью специальных символов:

- `>`: перенаправляет стандартный вывод (stdout) в файл, перезаписывая его, если он существует.
- `>>`: перенаправляет stdout в файл, добавляя данные в конец файла.
- `<`: направляет содержимое файла на стандартный ввод (stdin) команды.
- `2>`: перенаправляет стандартную ошибку (stderr) в файл (где 2 — дескриптор потока stderr).
- `2>&1`: перенаправляет поток stderr в поток stdout, объединяя их.

Это фундаментальный механизм для написания скриптов и логирования работы команд.
