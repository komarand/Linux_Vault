# Специальное устройство null

Файл `/dev/null` — это специальное устройство ядра, которое действует как "черная дыра" для данных.

Любые данные, записанные в `/dev/null`, мгновенно уничтожаются (отбрасываются) ядром. При чтении из него немедленно возвращается символ конца файла (EOF).

Оно повсеместно используется в конвейерах и скриптах для перенаправления нежелательного вывода (часто стандартных ошибок). Пример: `command > /dev/null 2>&1` полностью подавляет весь вывод команды.
