# Команда dd

Утилита `dd` (data description или disk dump) — мощный инструмент для низкоуровневого побайтового копирования данных.

Она часто используется для копирования данных напрямую между файлами устройств в `/dev`, минуя файловую систему. Например, `dd` может создать точный образ целого жесткого диска или раздела (`if=/dev/sda of=image.img`) или записать загрузочный ISO-образ на USB-накопитель.

Синтаксис `if=` (input file) и `of=` (output file) отличается от других команд Linux, а неправильное указание `of=` может мгновенно уничтожить данные на диске.
