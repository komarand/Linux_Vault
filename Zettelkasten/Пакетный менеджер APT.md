# Пакетный менеджер APT

APT (Advanced Package Tool) — это высокоуровневый менеджер пакетов, используемый в дистрибутивах на базе Debian (включая Ubuntu).

Он работает поверх низкоуровневой утилиты `dpkg`.
Основные команды:
- `apt update`: обновляет локальный кэш (список доступных пакетов) из репозиториев.
- `apt upgrade`: обновляет все установленные в системе пакеты до последних версий.
- `apt install package`: скачивает и устанавливает программу вместе со всеми зависимостями.
- `apt remove package`: удаляет программу (но может оставить её конфигурационные файлы).
- `apt search keyword`: ищет пакеты по ключевому слову в кэше.
