# Параметры командной строки ядра

При запуске загрузчик передает ядру набор текстовых параметров, определяющих его работу (Kernel Command Line Parameters).

Они конфигурируются в настройках загрузчика (например, в GRUB). Некоторые критически важные параметры:
- `root=`: указывает ядру, на каком устройстве (разделе или UUID) находится реальная корневая файловая система.
- `ro`: монтирует корневую ФС изначально в режиме "только чтение" (для безопасной проверки `fsck`).
- `quiet`: подавляет вывод большинства диагностических сообщений ядра на экран при загрузке.
- `init=`: позволяет указать нестандартную программу инициализации (вместо systemd).
