# Таблица разделов GPT

Таблица разделов GPT (GUID Partition Table) — это современный стандарт разметки дисков, являющийся частью стандарта UEFI.

Преимущества перед MBR:
1. Поддержка дисков размером до 9.4 Зеттабайт (ЗБ).
2. Поддержка до 128 первичных разделов по умолчанию (отпадает необходимость в расширенных и логических разделах).
3. Встроенная избыточность: копия таблицы разделов хранится в конце диска для восстановления при повреждениях.
4. Проверка целостности данных с помощью контрольных сумм CRC32.

GPT является стандартом де-факто для современных серверов и ПК.
