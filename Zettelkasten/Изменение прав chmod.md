# Изменение прав chmod

Утилита `chmod` (change mode) изменяет права доступа к файлам и каталогам.

Она поддерживает два синтаксиса:
1. **Символьный**: `chmod u+x file` (добавить владельцу право на выполнение), `chmod g-w file` (убрать у группы право на запись), `chmod o=r file` (остальным только чтение).
2. **Восьмеричный (числовой)**: `chmod 755 file` (Владелец: 4+2+1=7 (rwx), Группа: 4+1=5 (r-x), Остальные: 4+1=5 (r-x)).

Права можно изменять рекурсивно для всех файлов в каталоге с помощью ключа `-R`.
