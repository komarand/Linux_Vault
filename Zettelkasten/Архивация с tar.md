# Архивация с tar

Утилита `tar` (tape archive) используется для объединения нескольких файлов и каталогов в один архивный файл.

Изначально она была создана для записи на магнитные ленты, поэтому она не сжимает данные по умолчанию (получается просто большой файл `.tar`).

Частые комбинации команд:
- `tar -cvf archive.tar dir/` — создать (c) архив с выводом процесса (v) в файл (f).
- `tar -xvf archive.tar` — извлечь (x) файлы из архива.
- `tar -tvf archive.tar` — просмотреть список файлов (t) внутри архива без извлечения.
