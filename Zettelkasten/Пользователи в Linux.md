# Пользователи в Linux

Linux является многопользовательской операционной системой.

Каждый процесс и файл связаны с определенным пользователем, что лежит в основе системы безопасности и разграничения прав доступа. У каждого пользователя есть уникальный идентификатор (UID) и домашний каталог.

Большинство пользователей — это не живые люди, а системные пользователи, созданные для запуска демонов и служб в ограниченной среде (изоляции). Информацию о пользователях хранят файлы `/etc/passwd` и `/etc/shadow`.
