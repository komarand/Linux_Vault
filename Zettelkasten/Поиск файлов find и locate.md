# Поиск файлов find и locate

Для поиска файлов в файловой системе используются утилиты `find` и `locate`.

- Команда `find` выполняет поиск в реальном времени, обходя каталоги начиная с указанного. Она может искать по имени, размеру, времени модификации, правам доступа и другим критериям. Поддерживает выполнение команд над найденными файлами через параметр `-exec`.
- Команда `locate` использует заранее созданную базу данных (индекс) для поиска файлов по имени, что делает её намного быстрее `find`. Однако её база данных обновляется периодически (например, утилитой `updatedb`), поэтому она может не находить недавно созданные файлы.
